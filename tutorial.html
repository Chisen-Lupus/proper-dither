This section provides a step-by-step tutorial on how to use the proper-dither package.

  4. Speed Comparison: Compare the performance of the updated code with the legacy version. - 2.1-speed-comparison-of-updated-code.ipynb

  5. Simulated PSF Comparisons: Use simulated data from WebbPSF. - 3-compare-using-simulated-webbpsf.ipynb
